Absolute x-Path:

->It Starts with single forward slash (/)

->It Starts from html tag.(ROOT TAG)

Demo site: http://demo.guru99.com/test/selenium-xpath.html

Ex: /html/body/div[2]/div[1]/div/h4[1]/b[1]
